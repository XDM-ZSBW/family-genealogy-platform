üîó Naming Conventions

File Naming Patterns

  • People: firstname_lastname.md or firstname.md
  • Example: gladys_klingenberg.md, annie.md
  • Lowercase with underscores

  • No special characters (periods, commas removed)

  • Images: page_N_img_M.png and page_N_img_M.md

  • N = page number (1-indexed)
  • M = image index on that page (1-indexed)
  • Example: page_13_img_1.png

Wikilink Patterns

  • Internal links: Display Name
  • People links: Person Name
  • Image links: Image from Page N
  • Navigation links: Wiki Home
